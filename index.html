LATEST WORK

Phoney

Featured on TechCrunch and IBM Watson Blog. A mobile-first app that helps users get out of tough conversations. Utilizes IBM Watson API set to listen to a preset word or phrase. When user speaks the phrase, the app sends a call to the user's phone, using the Twilio API, giving the user a smooth excuse to exit.

Tech: Javascript, Node.js

Eclair

Created at Health 2.0 codeathon, with Eclair, we created a proactive way for seniors to EASILY avoid their MEDICARE PART D coverage gap! Eclair utilizes the Pokitdok Pharmacy Formulary API to look up costs of medications using the patient's insurance and the FBD Cloud Connector API to return alternative medication suggestions.

Tech: Ruby on Rails, Javascript, jQuery, Bootstrap.

Guardian

DBC Final Project. Guardian is a mobile-first(only) app that utilizes a Parrot AR drone 2.0 as an API to help get you home safely. When the the app is engaged, the user's location and the drone's video feed is sent to a contact so that they can watch over their friend.

Tech: Parrot AR Drone 2.0, Ruby on Rails, Javascript, jQuery, Flexbox.
